# Week11TuitionCalculator
A command-line program that calculates a student's tuition based on the number of credits they are enrolled in. Full-time students (12+ credits) pay a flat rate of $20,000. Part-time students pay $1,200 plus $1,700 per credit. Invalid input is caught and an error message is displayed.
What I Learned

Writing and calling custom functions with return values
Tiered conditional logic using if/elif/else
Formatting numbers as currency with f-strings
Separating logic from user input using a main() function

Tools Used

Python 3

How to Run
Download the file and run the following command in your terminal:
python Week11Tuition.py
